สิ่งที่ต้องเตรียม:
  - USB Flash Drive 2 ตัว (4GB+, format ด้วย Rufus)
  - สาย LAN ต่อทุกเครื่อง (switch ในห้อง)
  - ไฟล์ chumchim-clonezilla.iso
  - ไฟล์ school-pc-optimize.bat (ถ้าจะลดขนาด image)

ดาวน์โหลดทั้งหมด:
  \\166.166.1.24\TeacherSpace\Software\ChumChim-Clonezilla\

============================================================
STEP 1: สร้าง USB Boot (ทำครั้งเดียว)
============================================================
ทำ 2 ตัวเหมือนกัน:

  1. เปิด Rufus (คลิกขวา > Run as Administrator)
  2. เลือก USB > SELECT > chumchim-clonezilla.iso
  3. ตั้งค่า:
     - Partition scheme: MBR
     - Target system: BIOS or UEFI
  4. กด START > เลือก "Write in DD Image mode"
  5. รอจนเสร็จ > ทำซ้ำกับ USB ตัวที่ 2

============================================================
STEP 2: เตรียมเครื่องต้นแบบ (ลดขนาด image)
============================================================
*** ทำบนเครื่องต้นแบบที่มี Windows + โปรแกรมครบแล้ว ***

  1. Copy "school-pc-optimize.bat" จาก server ไปเครื่องต้นแบบ
     (หรือ copy จาก USB ถ้า format แบบ GPT/ISO mode)
  2. คลิกขวา > Run as Administrator
  3. รอ 10-15 นาที (7 ขั้นตอนอัตโนมัติ):
     - ลบ temp, cache, bloatware
     - ปิด services ไม่จำเป็น
     - บีบอัด Windows (Compact OS)
     - ตั้งค่าภาษาไทย + English
     - ย่อ partition
     - Zero-fill พื้นที่ว่าง
  4. เสร็จแล้ว > เครื่องปิดอัตโนมัติ

  ผลลัพธ์: ขนาดลดลง 30-80 GB, Clone เร็วขึ้นมาก!

  *** ถ้าไม่รัน optimize เครื่องที่มี Adobe หลายตัว
      image อาจใหญ่ 50-100 GB ***

============================================================
STEP 3: Clone เครื่องต้นแบบ (เก็บ image)
============================================================

  เครื่อง B (จะเป็น server เก็บ image):
  1. เสียบ USB#1 + สาย LAN
  2. เปิดเครื่อง > boot จาก USB
  3. เลือก "3 LAN Server"
  4. รอจนขึ้น "LAN SERVER READY"
     จดชื่อ IP ไว้ (เช่น 192.168.x.x)

  เครื่อง A (ต้นแบบ):
  5. เสียบ USB#2 + สาย LAN
  6. เปิดเครื่อง > boot จาก USB
  7. เลือก "1 Clone this PC"
  8. ระบบหา LAN Server อัตโนมัติ
  9. ตั้งชื่อ image เช่น "Room301-Adobe"
  10. ใส่หมายเหตุ (กด OK ข้ามได้)
  11. ยืนยัน > รอ Clone เสร็จ
  12. เครื่อง A ปิดอัตโนมัติ
  13. ถอด USB#2 จากเครื่อง A

  *** image ถูกเก็บใน HDD เครื่อง B ***
  *** เครื่อง B ยังเปิดอยู่ พร้อม Install ***

============================================================
STEP 4: Install ทุกเครื่องในห้อง
============================================================

--- วิธี A: PXE Boot (แนะนำ, ไม่ต้อง USB) ---

  เครื่อง B ยังเปิดอยู่ (LAN Server + PXE)

  เครื่อง C, D, E... ทุกเครื่อง:
  1. ต้องต่อสาย LAN
  2. เข้า BIOS > เปิด Network/PXE Boot (ทำครั้งเดียว)
     - กด F2/DEL/F12 ตอนเปิดเครื่อง
     - หา Boot > Network Boot > Enable
     - ตั้ง Network Boot เป็นลำดับแรก
     - Save & Exit
  3. เครื่องจะ boot จาก LAN อัตโนมัติ
  4. เห็น "ChumChim-Clonezilla (PXE)" > กด Enter
  5. Install อัตโนมัติ ~5 นาที
  6. เสร็จ > เครื่องปิดเอง
  7. เข้า BIOS อีกครั้ง > เปลี่ยน Boot Order กลับเป็น HDD
  8. เปิดเครื่อง > Windows พร้อมใช้!

--- วิธี B: USB Boot (ถ้า PXE ไม่ได้) ---

  1. เสียบ USB#2 เข้าเครื่อง C
  2. Boot จาก USB
  3. เลือก "2 Install to PC"
  4. ระบบหา LAN Server อัตโนมัติ
  5. Install ~5 นาที > เสร็จ
  6. ถอด USB#2 > เสียบเครื่อง D > ทำซ้ำ

============================================================
STEP 5: หลังติดตั้งเสร็จ
============================================================

  ทุกเครื่องจะมี:
  - Windows + โปรแกรมเหมือนต้นแบบ
  - ชื่อเครื่อง + IP ไม่ซ้ำกัน (ถ้า Sysprep แล้ว)
  - Auto-activate Windows (ต้องต่อ internet)

  เปลี่ยน BIOS กลับเป็น HDD boot:
  - เข้า BIOS > Boot Order > HDD/SSD first
  - Save & Exit
  - เปิดเครื่อง > Windows พร้อมใช้!

สำหรับอาจารย์: แช่เครื่อง (UWF)
============================================================

  เปิดแช่ (Freeze):
  1. ดับเบิลคลิก "System Settings" บน Desktop
  2. ใส่รหัส: teacher
  3. เลือก 1 > เครื่อง restart > แช่แล้ว
  4. เด็กใช้อะไรก็ได้ restart = กลับเดิม

  ปิดแช่ (Unfreeze) เพื่อลงโปรแกรมเพิ่ม:
  1. ดับเบิลคลิก "System Settings"
  2. ใส่รหัส: teacher
  3. เลือก 2 > restart > ลงโปรแกรมได้
  4. เสร็จแล้ว > Freeze กลับ

แก้ปัญหา
============================================================

ถาม: เครื่องไม่ boot จาก USB?
ตอบ: เข้า BIOS > ปิด Secure Boot > USB first

ถาม: PXE Boot ไม่ขึ้น?
ตอบ: เข้า BIOS > เปิด Network/PXE Boot > ต่อสาย LAN

ถาม: PXE boot แล้วขึ้น "boot image not found"?
ตอบ: รอ Server READY ก่อน แล้วค่อยเปิดเครื่อง Client

ถาม: Clone ช้ามาก?
ตอบ: รัน school-pc-optimize.bat ก่อน Clone

ถาม: Install เสร็จแต่เครื่องไม่ปิด?
ตอบ: PXE boot วน loop — เปลี่ยน BIOS เป็น HDD boot

ถาม: ไม่มี school-pc-optimize.bat?
ตอบ: โหลดจาก server หรือ GitHub
