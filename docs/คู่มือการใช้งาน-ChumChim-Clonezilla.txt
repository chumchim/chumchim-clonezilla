============================================================
  ChumChim-Clonezilla v3.0
  เครื่องมือ Clone และ Deploy PC สำหรับห้องเรียน
============================================================

หลักการทำงาน:
  Clone PC 1 เครื่อง -> นำไปลงได้หลายเครื่อง
  1. ติดตั้งโปรแกรมบนเครื่องต้นแบบ
  2. Clone เครื่องต้นแบบ (สร้าง image)
  3. นำ image ไปลงเครื่องอื่นๆ
  4. เสร็จ!

============================================================
สิ่งที่ต้องเตรียม
============================================================
- USB Flash Drive (แนะนำ 32GB ขึ้นไป, ขั้นต่ำ 4GB)
- โปรแกรม Rufus (https://rufus.ie) — ไม่ต้องติดตั้ง
- ไฟล์ chumchim-clonezilla.iso

ดาวน์โหลดทั้งหมดได้จาก:
  \\166.166.1.24\TeacherSpace\Software\ChumChim-Clonezilla\

============================================================
ขั้นตอนที่ 1: สร้าง USB Boot
============================================================
มี 2 วิธี เลือกตามประเภทเครื่อง:

--- วิธี A: ใช้ได้ทุกเครื่อง (แนะนำ) ---
  ใช้ DD Image mode รองรับทั้งเครื่องเก่า (BIOS) และใหม่ (UEFI)
  หมายเหตุ: USB จะไม่แสดงใน File Explorer หลังสร้าง
            ระบบจะสร้างพื้นที่เก็บ image อัตโนมัติตอน boot ครั้งแรก

  1. เปิด Rufus (คลิกขวา > Run as Administrator)
  2. Device: เลือก USB drive ที่ต้องการ
  3. SELECT > เลือกไฟล์ chumchim-clonezilla.iso
  4. ตั้งค่า:
     - Partition scheme: MBR
     - Target system: BIOS or UEFI
     - File system: FAT32
  5. กด START
  6. เมื่อถาม mode ให้เลือก "Write in DD Image mode"
     (ถ้าขึ้น "Failed to download file" ให้กด Close
      แล้วกด START ใหม่ เลือก DD mode)
  7. รอจนเสร็จ (~2-5 นาที)

--- วิธี B: เฉพาะเครื่องใหม่ (UEFI) ---
  USB แสดงใน File Explorer ได้ เก็บ image บน USB ตัวเดียวได้

  1. เปิด Rufus (คลิกขวา > Run as Administrator)
  2. Device: เลือก USB drive (แนะนำ 16GB ขึ้นไป)
  3. SELECT > เลือกไฟล์ chumchim-clonezilla.iso
  4. ตั้งค่า:
     - Partition scheme: GPT
     - Target system: UEFI
     - File system: FAT32
  5. กด START > เลือก "Write in ISO Image mode"
  6. รอจนเสร็จ

============================================================
ขั้นตอนที่ 2: Clone เครื่องต้นแบบ (สร้าง Image)
============================================================
ใช้สำหรับ "สำเนา" เครื่องที่ติดตั้งโปรแกรมเสร็จแล้ว

  1. เสียบ USB Boot เข้าเครื่องต้นแบบ
  2. เปิดเครื่อง กด F12 (หรือ F2/DEL) เลือก Boot จาก USB
  3. รอจนเข้าเมนู ChumChim-Clonezilla v3.0
  4. เลือก "1 Clone this PC"
  5. เลือกโหมด:

  --- Auto (แนะนำ) ---
     ระบบตรวจหา disk ต้นทาง + ที่เก็บ ให้อัตโนมัติ
  6. ตั้งชื่อ Image เช่น "Room301-Win11-Office"
  7. ใส่หมายเหตุ (กด OK ข้ามได้)
     เช่น "Win11 + Office + Adobe"
  8. ดูสรุป > กด Yes เพื่อเริ่ม
  9. รอ ~10-30 นาที (ขึ้นอยู่กับขนาด disk)
  10. "CLONE COMPLETE!" แสดงขนาด image ที่ได้

  --- Manual ---
     เลือกเองทุกขั้นตอน (สำหรับผู้ใช้ขั้นสูง)
  6. เลือก disk ต้นทาง
     - [WIN] = disk ที่มี Windows
  7. เลือก disk สำหรับเก็บ image
     - [BOOT] = USB ที่ใช้ boot
     - USB, External HDD, หรือ HDD ลูกอื่นในเครื่อง
     * ห้ามเลือก disk เดียวกับต้นทาง (ระบบจะเตือน)
     * ถ้าพื้นที่ไม่พอ ระบบจะเตือน
  8. เลือก partition
  9. ตั้งชื่อ Image
  10. ใส่หมายเหตุ (กด OK ข้ามได้)
  11. เลือกความเร็ว:
      - Fast   = เร็ว ไฟล์ใหญ่กว่า
      - Normal = สมดุล (แนะนำ)
      - Small  = ไฟล์เล็ก แต่ช้า
  12. ดูสรุป > กด Yes > รอจนเสร็จ

============================================================
ขั้นตอนที่ 3: ลง Image ไปเครื่องอื่น (Install)
============================================================
ใช้สำหรับนำ image ที่ Clone ไว้ ไปลงเครื่องอื่น

  1. เสียบ USB Boot เข้าเครื่องเป้าหมาย
     (ถ้า image อยู่ USB/HDD อีกตัว ให้เสียบด้วย)
  2. เปิดเครื่อง Boot จาก USB
  3. เลือก "2 Install to PC"
  4. เลือกโหมด:

  --- Auto (แนะนำ) ---
     ระบบหา disk เป้าหมายให้อัตโนมัติ
     (เลือก disk ที่ใหญ่ที่สุด ข้าม USB)
  5. เลือก image ที่ต้องการลง
  6. ดูสรุป — ตรวจสอบชื่อ disk ให้ถูกต้อง
     *** ข้อมูลใน disk เป้าหมายจะถูกลบทั้งหมด! ***
  7. กด Yes > รอ ~10-20 นาที
  8. "INSTALL COMPLETE!" > ถอด USB > Restart
  9. ทำซ้ำกับเครื่องถัดไปได้เลย (image ไม่ถูกลบ)

  --- Manual ---
  5. เลือก image
  6. เลือก disk เป้าหมายเอง
  7. ยืนยัน > รอจนเสร็จ

============================================================
ขั้นตอนที่ 4: Deploy หลายเครื่องพร้อมกัน (LAN Multicast)
============================================================
ลง image ทีละหลายเครื่องผ่านสาย LAN (ทดลอง)

  1. Boot USB บนเครื่องหลัก (เครื่องที่มี image)
  2. เลือก "3 Deploy to many PCs (LAN Multicast)"
  3. เลือก image ที่ต้องการส่ง
  4. ตั้งจำนวนเครื่องที่จะรับ
  5. เปิดเครื่องอื่นๆ ให้ Boot จาก LAN (PXE Boot)
  6. เครื่องอื่นจะรับ image อัตโนมัติ

  หมายเหตุ: ต้องต่อสาย LAN ทุกเครื่อง
  เครื่องรับต้องเปิด PXE Boot ใน BIOS

============================================================
ขั้นตอนที่ 5: จัดการ Image
============================================================
เลือก "4 Manage Images" จากเมนูหลัก:

  1. View all images — ดู image ทั้งหมด (ชื่อ, ขนาด, วันที่, หมายเหตุ)
  2. Copy image — คัดลอก image ไป USB/HDD อีกตัว
  3. Delete an image — ลบ image
  4. Add/edit note — เพิ่ม/แก้ไขหมายเหตุ

--- Copy Image (คัดลอก image) ---
  ใช้สำหรับ: Clone ครั้งเดียว แล้ว copy image ไปหลาย USB/HDD
  ไม่ต้องนั่ง Clone ซ้ำอีก!

  1. เลือก "2 Copy image to another disk"
  2. เลือก image ที่ต้องการ copy
  3. เลือก disk/partition ปลายทาง
     (เสียบ USB หรือ External HDD อีกตัว)
  4. ดูสรุป > กด Yes
  5. รอจน copy เสร็จ
  6. USB/HDD ตัวใหม่พร้อม Install ได้เลย

เมนูอื่นๆ

  - "5 Help"     = วิธีใช้งานเบื้องต้น
  - "0 Shutdown" = ปิดเครื่อง

คำแนะนำ
============================================================
  - ใช้ USB 3.0 จะเร็วกว่ามาก (เสียบ port สีฟ้า)
  - ตั้งชื่อ image ให้สื่อความหมาย เช่น ห้อง+โปรแกรม+วันที่
    ตัวอย่าง: "Room301-Win11-Office-Apr07"
  - Clone จากเครื่องที่ setup เสร็จสมบูรณ์แล้ว
  - ทดสอบ Install ลง 1 เครื่องก่อน แล้วค่อยทำทั้งห้อง
  - USB ตัวเดียวเก็บ image ได้หลายตัว
  - ลงซ้ำกี่เครื่องก็ได้ image ไม่ถูกลบ
  - แนะนำใช้ Auto mode สำหรับผู้ใช้ทั่วไป
  - ใช้ Manual mode ถ้าต้องการควบคุมเอง
  - Auto mode เลือก compression อัตโนมัติตามพื้นที่ว่าง:
    พื้นที่เหลือเยอะ = เร็วสุด / พื้นที่น้อย = บีบอัดมากขึ้น
  - ระบบแสดงจำนวน partition ก่อนเริ่ม Clone/Install
    เช่น "Partitions: 3" หมายความว่าจะทำ 3 รอบ ซึ่งเป็นปกติ

============================================================
แก้ปัญหาเบื้องต้น
============================================================

ถาม: เครื่องไม่ Boot จาก USB?
ตอบ: เข้า BIOS > ปิด Secure Boot > ให้ USB ขึ้นก่อน

ถาม: ขึ้น "BIOS/LEGACY BOOT OF UEFI-ONLY MEDIA"?
ตอบ: USB ถูกสร้างแบบ UEFI แต่เครื่องนี้ใช้ Legacy
     แก้: ใช้ Rufus สร้างใหม่ ด้วย MBR + DD Image mode

ถาม: ขึ้น "device descriptor read/64, error -32" ซ้ำๆ?
ตอบ: USB port อ่าน drive ไม่ได้ ลองตามลำดับ:
     1. ย้าย USB ไปเสียบ port ด้านหลังเครื่อง
     2. ลอง port USB 2.0 (สีดำ) แทน 3.0 (สีฟ้า)
     3. ลอง USB drive ตัวอื่น
     4. BIOS > ลอง disable xHCI/USB 3.0

ถาม: จอค้างที่ "hub doesn't have any ports"?
ตอบ: ไม่ใช่ error จริง รอ ~30 วินาที
     ถ้าค้างเกิน 1 นาที ลองเปลี่ยน USB port

ถาม: "No writable disk found for saving image"?
ตอบ: ไม่พบที่เก็บ image ตรวจสอบ:
     - วิธี A (DD): ระบบสร้างพื้นที่เก็บอัตโนมัติตอน boot
       ถ้า USB เล็กเกิน ให้เสียบ USB/HDD อีกตัว
     - วิธี B (ISO): ตรวจว่า USB มีพื้นที่ว่างพอ

ถาม: Clone/Install ช้ามาก?
ตอบ: ใช้ USB 3.0 เสียบ port สีฟ้า

ถาม: Install เสร็จแต่เครื่อง Boot ไม่ขึ้น?
ตอบ: เข้า BIOS > เปลี่ยน Boot Order ให้ HDD/SSD ขึ้นก่อน

ถาม: ไม่เห็น disk ในเมนู?
ตอบ: disk อาจเป็น RAID mode > เข้า BIOS เปลี่ยนเป็น AHCI

ถาม: Clone ล้มเหลว?
ตอบ: รายละเอียด error แสดงในหน้า dialog
     สาเหตุที่พบบ่อย:
     - พื้นที่เต็ม: ใช้ USB/HDD ที่ใหญ่กว่า
     - disk ว่าง: เครื่องต้นทางต้องมี Windows ติดตั้ง
     ลอง Manual mode ถ้า Auto mode ไม่ได้

ถาม: เก็บ image ใน USB boot ได้มั้ย?
ตอบ: วิธี A (DD): ได้ ระบบสร้างพื้นที่ให้อัตโนมัติ
     วิธี B (ISO): ได้ เลือก USB ที่มี [BOOT] กำกับ

ถาม: เก็บ image ลง HDD ในเครื่องได้มั้ย?
ตอบ: ได้ ถ้าเครื่องมี disk 2 ลูก (เช่น SSD + HDD)
     Clone จากลูกหนึ่ง เก็บ image อีกลูกหนึ่ง
