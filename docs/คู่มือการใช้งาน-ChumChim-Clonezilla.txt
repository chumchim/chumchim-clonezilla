============================================================
  ChumChim-Clonezilla v3.0
  เครื่องมือ Clone และ Deploy PC สำหรับห้องเรียน
============================================================

หลักการทำงาน:
  Clone PC 1 เครื่อง -> นำไปลงได้หลายเครื่อง
  1. ติดตั้งโปรแกรมบนเครื่องต้นแบบ
  2. รัน school-pc-optimize.bat (เตรียมเครื่อง)
  3. Clone เครื่องต้นแบบ (สร้าง image)
  4. นำ image ไปลงเครื่องอื่นๆ
  5. เสร็จ!

============================================================
สิ่งที่ต้องเตรียม
============================================================
- USB Flash Drive 2 ตัว (4GB ขึ้นไป สำหรับ boot)
  หรือ USB 1 ตัว 128GB+ (boot + เก็บ image)
- โปรแกรม Rufus — ไม่ต้องติดตั้ง
- ไฟล์ chumchim-clonezilla.iso

ดาวน์โหลดทั้งหมดได้จาก:
  \\166.166.1.24\TeacherSpace\Software\ChumChim-Clonezilla\

ไฟล์ทั้งหมดบน server:
  chumchim-clonezilla.iso  = ไฟล์สำหรับสร้าง USB Boot
  rufus.exe                = โปรแกรมเขียน ISO ลง USB
  school-pc-optimize.bat   = เตรียมเครื่องก่อน Clone
  school-pc-sysprep.bat    = Reset เครื่องก่อน Clone (ขั้นสุดท้าย)

============================================================
ขั้นตอนที่ 1: สร้าง USB Boot
============================================================
  1. เปิด Rufus (คลิกขวา > Run as Administrator)
  2. Device: เลือก USB drive
  3. SELECT > เลือกไฟล์ chumchim-clonezilla.iso
  4. ตั้งค่า:
     - Partition scheme: MBR
     - Target system: BIOS or UEFI
     - File system: FAT32
  5. กด START
  6. เมื่อถาม mode ให้เลือก "Write in DD Image mode"
  7. รอจนเสร็จ (~2-5 นาที)

  หมายเหตุ: USB ที่สร้างด้วย DD mode จะไม่แสดงใน File Explorer
  ระบบจะสร้างพื้นที่เก็บ image อัตโนมัติตอน boot ครั้งแรก
  ใช้ได้ทั้งเครื่องเก่า (BIOS) และเครื่องใหม่ (UEFI)

============================================================
ขั้นตอนที่ 2: เตรียมเครื่องต้นแบบ
============================================================
ทำบนเครื่องต้นแบบที่ติดตั้ง Windows + โปรแกรมเสร็จแล้ว

  ขั้นตอนที่ 2.1: Optimize เครื่อง
  1. คัดลอก school-pc-optimize.bat ไปเครื่องต้นแบบ
  2. คลิกขวา > Run as Administrator
  3. รอ 10-15 นาที (ทำงาน 7 ขั้นตอนอัตโนมัติ)
     - ลบ temp, cache, bloatware
     - ปิด service ไม่จำเป็น
     - บีบอัด Windows (Compact OS)
     - ตั้งค่าภาษา ไทย+English, timezone ไทย
     - ย่อ partition + zero-fill
  4. Restart เครื่อง

  ผลลัพธ์: ขนาดลดลง 30-80 GB, เครื่องเร็วขึ้น

  ขั้นตอนที่ 2.2: Sysprep (ถ้าต้องการ)
  *** ทำเป็นขั้นตอนสุดท้ายเท่านั้น ***
  1. คัดลอก school-pc-sysprep.bat ไปเครื่องต้นแบบ
  2. คลิกขวา > Run as Administrator
  3. เครื่องจะปิดอัตโนมัติหลัง Sysprep เสร็จ
  4. ห้ามเปิดเครื่องอีก — ไปขั้นตอน Clone เลย

  Sysprep ทำอะไร:
  - Reset ชื่อเครื่อง (แต่ละเครื่องจะได้ชื่อใหม่)
  - Reset SID (ไม่ชนกันบน network)
  - เตรียมรองรับ hardware ต่าง

============================================================
ขั้นตอนที่ 3: Clone เครื่องต้นแบบ
============================================================
มี 2 วิธี:

--- วิธี A: Clone ผ่าน LAN (เร็ว, แนะนำ) ---
  ต้องการ: USB Boot 2 ตัว + สาย LAN

  เครื่อง B (เก็บ image):
  1. เสียบ USB Boot + สาย LAN
  2. Boot จาก USB
  3. เลือก "5 LAN Server"
  4. ระบบตั้ง server อัตโนมัติ
  5. แสดง "Server ready. IP: xxx.xxx.xxx.xxx"

  เครื่อง A (ต้นแบบ):
  1. เสียบ USB Boot + สาย LAN
  2. Boot จาก USB
  3. เลือก "1 Clone this PC" > Auto
  4. ระบบหา LAN Server อัตโนมัติ
  5. ตั้งชื่อ image + หมายเหตุ
  6. ยืนยัน > รอ ~4-10 นาที
  7. Clone เสร็จ! image อยู่ในเครื่อง B

--- วิธี B: Clone ลง USB/External HDD ---
  ต้องการ: USB Boot + USB/HDD สำหรับเก็บ image

  1. เสียบ USB Boot + USB/HDD เข้าเครื่องต้นแบบ
  2. Boot จาก USB
  3. เลือก "1 Clone this PC" > Auto
  4. ระบบหา USB/HDD อัตโนมัติ
  5. ตั้งชื่อ image + หมายเหตุ
  6. ยืนยัน > รอ ~10-30 นาที
  7. Clone เสร็จ! image อยู่ใน USB/HDD

============================================================
ขั้นตอนที่ 4: ลง image ไปเครื่องอื่น (Install)
============================================================
--- ถ้าใช้ LAN Server (วิธี A ข้างบน) ---
  เครื่อง B ยังเปิดอยู่ (LAN Server)

  เครื่อง C, D, E... (ทีละเครื่อง):
  1. เสียบ USB Boot + สาย LAN
  2. Boot จาก USB
  3. เลือก "2 Install to PC" > Auto
  4. ระบบหา LAN Server + เลือก image อัตโนมัติ
  5. ยืนยัน > รอ ~4-10 นาที
  6. Install เสร็จ! ถอด USB > Restart
  7. ทำซ้ำกับเครื่องถัดไป (image ไม่ถูกลบ)

--- ถ้าใช้ USB/HDD (วิธี B ข้างบน) ---
  1. เสียบ USB Boot + USB/HDD ที่มี image
  2. Boot จาก USB
  3. เลือก "2 Install to PC" > Auto
  4. ระบบหา image + เลือก disk เป้าหมายอัตโนมัติ
  5. ยืนยัน > รอ ~10-20 นาที
  6. Install เสร็จ! ถอด USB > Restart

  *** ข้อมูลใน disk เป้าหมายจะถูกลบทั้งหมด! ***

============================================================
ขั้นตอนที่ 5: จัดการ Image
============================================================
เลือก "4 Manage Images" จากเมนูหลัก:

  1. View all images — ดู image ทั้งหมด
  2. Copy image — คัดลอก image ไป USB/HDD อีกตัว
     (Clone ครั้งเดียว แล้ว copy ไปหลาย USB ได้)
  3. Delete an image — ลบ image
  4. Add/edit note — เพิ่ม/แก้ไขหมายเหตุ

============================================================
เมนูอื่นๆ
============================================================
  - "3 Deploy to many PCs" = Multicast ผ่าน LAN (ลงพร้อมกันหลายเครื่อง)
  - "5 LAN Server" = เปิดเครื่องเป็น server สำหรับ Clone/Install ผ่าน LAN
  - "6 Help" = วิธีใช้งานเบื้องต้น
  - "0 Shutdown" = ปิดเครื่อง

============================================================
คำแนะนำ
============================================================
  - ใช้ USB 3.0 จะเร็วกว่ามาก (เสียบ port สีฟ้า)
  - Clone ผ่าน LAN เร็วกว่า USB 5 เท่า (ถ้า LAN 1 Gbps)
  - ตั้งชื่อ image ให้สื่อความหมาย
    ตัวอย่าง: "Room301-Win11-Office-Apr07"
  - รัน school-pc-optimize.bat ก่อน Clone ทุกครั้ง
  - ทดสอบ Install ลง 1 เครื่องก่อน แล้วค่อยทำทั้งห้อง
  - Auto mode เลือก compression ตามพื้นที่ว่างอัตโนมัติ
  - ระบบแสดงจำนวน partition ก่อนเริ่ม Clone/Install
  - กด ESC หรือ Cancel ได้ตลอด กลับเมนูหลักเสมอ

============================================================
แก้ปัญหาเบื้องต้น
============================================================

ถาม: เครื่องไม่ Boot จาก USB?
ตอบ: เข้า BIOS > ปิด Secure Boot > ให้ USB ขึ้นก่อน

ถาม: ขึ้น "BIOS/LEGACY BOOT OF UEFI-ONLY MEDIA"?
ตอบ: ใช้ Rufus สร้างใหม่ด้วย MBR + DD Image mode

ถาม: ขึ้น "device descriptor read/64, error -32"?
ตอบ: ย้าย USB ไป port ด้านหลังเครื่อง หรือลอง USB ตัวอื่น

ถาม: "No writable disk found"?
ตอบ: เสียบ USB/HDD สำหรับเก็บ image
     หรือเปิด LAN Server บนเครื่องอื่น

ถาม: Clone ล้มเหลว?
ตอบ: รายละเอียดแสดงในหน้า dialog
     สาเหตุที่พบบ่อย:
     - พื้นที่เต็ม: ใช้ USB/HDD ที่ใหญ่กว่า
     - disk ว่าง: ต้องมี Windows ติดตั้ง
     - รัน school-pc-optimize.bat ก่อน Clone

ถาม: Clone/Install ช้ามาก?
ตอบ: 1. รัน school-pc-optimize.bat ก่อน (ลดขนาด 30-80 GB)
     2. ใช้ LAN แทน USB (เร็วขึ้น 5 เท่า)
     3. ใช้ USB 3.0 + port สีฟ้า

ถาม: Install เสร็จแต่ Boot ไม่ขึ้น?
ตอบ: เข้า BIOS > Boot Order ให้ HDD/SSD ขึ้นก่อน

ถาม: ไม่เห็น disk ในเมนู?
ตอบ: BIOS > เปลี่ยน RAID เป็น AHCI
